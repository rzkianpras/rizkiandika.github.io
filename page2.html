KELAS BERPRESTASI

Tujuan dari kursus robotik ini adalah siswa siap mengikuti berbagai lomba atau kontes robot.

Maze Solving Robot Contest (with AVR)

Adalah program kursus robotik yang khusus mempersiapkan peserta didik untuk mengikuti lomba robot atau olimpiade robot kategori maze solving robot dengan mikrokontroller berbasis AVR.

Transporter Robot Contest

Adalah program kursus robotik dimana siswa dipersiapkan khusus untuk mengikuti lomba robot atau olimpiade robot dalam kategori transporter robot.

Maze Solving Robot Contest (with Arduino)

Adalah program kursus robotik yang khusus mempersiapkan peserta didik untuk mengikuti lomba robot atau olimpiade robot kategori maze solving robot dengan mikrokontroller berbasis Arduino.

Sumo Robot Contest

Adalah program kursus robotik dimana peserta didik dipersiapkan khusus untuk mengikuti lomba robot atau olimpiade robot dalam kategori sumo robot.
